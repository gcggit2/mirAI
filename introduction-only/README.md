# mirAI塾 ランディングページ

## プロジェクト概要

**mirAI塾**は、生成AIを実務レベルで活用したい方向けの月額980円の実践型オンラインコミュニティのランディングページです。

AIを"使う人"から"使いこなす人"へと成長できる環境を提供し、経営者・会社員・フリーランス・個人事業主など、あらゆる職種の方々が参加できるコミュニティです。

## 🎨 デザインコンセプト

### カラーパレット

- **メインカラー**: 白 (#ffffff) - 清潔感と信頼性
- **サブカラー**: 深い青 (#1d5a9f) - 専門性と信頼感
- **アクセントカラー**: オレンジ (#ea5d08) - 行動喚起とエネルギー
- **文字色**: ダークグレー (#333333) - 読みやすさ

### デザインの特徴

✨ **リッチでプロフェッショナルなデザイン**
- 絵文字を排除し、Font Awesomeアイコンを使用
- グラデーション効果で深みのあるビジュアル
- シャドウとアニメーションで立体感を演出
- モダンなタイポグラフィで視認性を向上

📱 **完全レスポンシブデザイン**
- デスクトップ、タブレット、スマートフォンに完全対応
- モバイル用ハンバーガーメニュー実装
- 768px以下で最適化されたレイアウト

⚡ **インタラクティブな体験**
- スムーススクロール
- スクロールフェードインアニメーション
- FAQアコーディオン
- 料金の数字カウントアップアニメーション
- ホバーエフェクト

## 🛠️ カスタマイズ方法

### 色の変更
`css/style.css`の`:root`セクションで変更可能:

```css
:root {
    --color-white: #ffffff;
    --color-blue: #1d5a9f;
    --color-orange: #ea5d08;
    --color-gray: #333333;
}
```

### フォントの変更
Google Fontsを使用。`index.html`の`<head>`内で変更:

```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;500;700;900&display=swap" rel="stylesheet">
```

### コンテンツの編集
`index.html`内の各セクションを直接編集してください。

### CTAボタンのURL変更
現在のURL: `https://forms.gle/koduezq4vVUTjv8x6`

すべてのCTAボタンのURLを変更する場合は、`index.html`内で検索置換してください。
