# mirAI塾 AIセミナー ランディングページ

## プロジェクト概要

「デキる経営者はなぜ、AIに仕事を任せるのか？」をテーマとした、mirAI塾のオンラインセミナー告知用ランディングページです。人手不足を解消し、生産性を劇的に向上させる「人間とAIの業務分担」について学べるセミナーの魅力を伝えます。

## 開発者向け情報

### ローカル開発

このプロジェクトは静的HTMLサイトです。ローカルサーバーで実行することをお勧めします：

```bash
# Python 3を使用する場合
python -m http.server 8000

# Node.jsのlive-serverを使用する場合
npx live-server
```

### カスタマイズ

- **カラー変更**: `css/style.css`の`:root`セクションで変数を編集
- **コンテンツ変更**: `index.html`で各セクションを編集
- **アニメーション調整**: `js/main.js`のobserverOptionsを編集

## パフォーマンス最適化

- 画像は外部URLから読み込み
- Google Fontsは最適化されたpreconnectを使用
